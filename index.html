ABOUT

SpookyOugi is an FTM token with oldschool mechanics and burnt liquidity.

Ticker: SPOOKY.

Contract: 0x3898c17f84f9fcb754ff8c7ffe9ed3382717667a

Supply is 1 billion spooky. 100k marketcap. Liquidity growing steadily, add to the pool, sit back, and accumulate some FTM and spooky.

Chart can be found on kek.tools or dex.zoocoin Invest responsibly, and dyor.

HOW TO BUY

  1. Install Metamask for Chrome or download Mobile App
  2. Setup custom RPC - Fantom tutorial
  3. Go to FTM - SPOOKY Pair
  4. Click connect wallet in upper left corner
  5. Make sure you have FTM in your wallet and buy
  6. If you need FTM for a transaction there is a faucet channel use the commands !faucet and !withdraw
